aniDEX - Anime Explorer

Project Overview

aniDEX is an anime explorer web application that allows users to search and discover anime series and movies. The application uses the Jikan API to fetch real-time anime data from MyAnimeList.

Users can search for specific anime, filter by type and genre, sort results, and save their favorite shows. The interface is designed to be intuitive and visually appealing for anime enjoyers and a more modern audience.

Requirements & Features

  • Uses Jikan API (specified API option)
  • Search functionality with real-time API calls
  • Implements 4 filter controls (Type, Genre, Sort By, Sort Order)
  • Saves user data in localStorage (search term, filters, favorites)
  • Favorite system with persistent storage
  • Includes pagination with next/previous buttons
  • Features loading spinner during API calls
  • Responsive design that works on different screen sizes
  • Detailed anime information in modal view
  • Pagination for browsing large result sets
  • Responsive grid layout for anime cards
  • Modern dark theme with smooth animations

API Used

The application uses the Jikan REST API v4, which is an unofficial MyAnimeList API. This API provides comprehensive anime data including titles, images, ratings, genres, and synopsis information.

Technical Implementation

The project uses vanilla JavaScript with modern ES6+ features. The code follows best practices with modular functions, proper error handling, and clean separation of concerns.

CSS Grid and Flexbox are used for responsive layouts. Local Storage API maintains user preferences between sessions. The application state management ensures consistent user experience.
